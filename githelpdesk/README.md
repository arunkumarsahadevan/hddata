
# Helpdesk Dashboard (GitHub Pages)

Static, client‑side dashboard (Plotly + native HTML table).

## Try it locally
Open `index.html` in a browser and use the file picker to load a JSON.

## Sample data
- `samples/helpdesk_sample_1200_tickets.json`
- `samples/helpdesk_sample_5000_tickets.json`

## Publish to GitHub Pages
- Push this repo to GitHub.
- In **Settings → Pages**, set **Source** to *Deploy from branch*.
- Select your default branch (e.g. `main`) and folder **`/ (root)`**.
- Your site will be available at `https://<your-user>.github.io/<repo-name>/`.

> `.nojekyll` is included to bypass Jekyll processing.
